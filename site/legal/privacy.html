隐私政策 · Privacy Policy

生效日期: 2026-06-06（公测期） 适用版本: KnowMe v0.5+ 联系方式: privacy@useknowme.com

一句话摘要

你的笔记永远在你的电脑上。我们没有任何远程服务器持续读取你的内容。

1. 我们是谁

KnowMe / 知我 由独立开发者 @Timeflys2018 开发与维护。 当前为个人项目，未注册公司主体。

如有隐私问题，请邮件 privacy@useknowme.com。

2. 我们收集什么数据

2.1 核心原则

KnowMe 是 本地优先 应用。默认情况下，我们不收集任何个人数据。

2.2 我们不收集

  • ❌ 你的笔记内容
  • ❌ 你的文件 / 文件夹结构
  • ❌ 你的搜索查询
  • ❌ 你的 AI 对话内容
  • ❌ 你的设备文件路径
  • ❌ 你的 IP 地址（除非你访问 useknowme.com 网站，那是 Cloudflare 自动记录）
  • ❌ 你的位置信息
  • ❌ 任何 cookie 用于跨站追踪

2.3 我们可能收集（仅在你显式触发时）

场景 收集什么 用途 存储
你注册 / 登录 KnowMe 账号 邮箱地址、验证码状态、会话 token、设备标识 登录、邮箱验真、找回密码、设备管理、未来权益校验 Cloudflare D1 / KV；refresh token 在本机安全存储
检查更新 App 版本号、操作系统类型 推送更新通知 不存储，仅一次性查询
你提交 bug 报告 你提交的内容（你自己决定附什么） 修 bug GitHub Issues 公开
你订阅邮件列表（如有） 邮箱地址 发产品更新 邮件服务商，可随时退订
你购买 License 邮箱地址、订单号、付款金额 发 license key、退款 License 数据库
你访问 useknowme.com / docs.useknowme.com IP、UA、访问页面（Cloudflare） 流量统计、防 DDoS Cloudflare 30 天

2.4 第三方 LLM API 调用（关键）

KnowMe 的 AI 功能（Wiki 编译、Dream-cycle 等）会用你的 API key调用第三方 LLM 服务（OpenAI / Anthropic / 智谱等）。

重要：当你调用 AI 时，
  发送：你的笔记内容片段（用于 LLM 推理）
  发送方：直接从你的电脑发送（不经过我们的服务器）
  接收方：你配置的 LLM 服务商
  我们：完全不可见，无法读取你与 LLM 的对话

这意味着： - ✅ 我们看不到你和 AI 的对话内容 - ⚠️ LLM 服务商（OpenAI 等）会看到你发送的内容 - ⚠️ 你需要自己阅读他们的隐私政策

3. 数据存储位置

3.1 你的笔记

存储位置：你电脑本地的 SQLite 数据库（.knowme/ 目录）
访问权限：仅你自己
我们的访问：✅ 零访问
备份：你自己负责（可用 Time Machine / iCloud Drive 等）

3.2 网站访问数据（Cloudflare）

useknowme.com / docs.useknowme.com / download.useknowme.com / api.useknowme.com
存储位置：Cloudflare 全球节点
保留时间：30 天（Cloudflare Web Analytics）
访问权限：仅 owner（@Timeflys2018）通过 Cloudflare Dashboard 查看聚合数据
我们的使用：仅用于了解流量趋势，不与个人关联

3.3 账号数据（Cloudflare D1 / KV）

api.useknowme.com
存储位置：Cloudflare D1 / KV
存储内容：邮箱、验证码状态、会话/设备记录、限流计数
不存储：明文密码、你的笔记内容、你的 AI 对话内容
用途：登录、邮箱验真、找回密码、设备吊销、未来权益校验

3.4 License 数据（Cloudflare KV）

存储位置：Cloudflare Workers KV（全球分布）
存储内容：邮箱、license key、SKU、激活设备指纹（哈希值）
保留时间：License 有效期内 + 退款后 90 天审计
我们的使用：仅用于校验 license 有效性

4. 数据共享

我们不与第三方共享你的数据，除了：

  1. 支付处理: 当你购买 license 时，邮箱和付款金额会与正式支付渠道（如 Paddle / Stripe / 国内商户收款渠道）共享；当你自愿赞助时，相关信息由赞助平台处理。这些渠道有自己的隐私政策。
  2. 法律要求: 如果中国法律 / 其他司法辖区强制要求，我们可能配合调查。我们会尽最大努力提前通知你。
  3. 业务转移: 如果项目被收购或转移，你的数据可能转移给受让方。届时会提前 90 天通知。

5. 你的权利

5.1 数据访问权

  • 笔记内容：你已经在你电脑上，无需向我们请求
  • 账号 / License 信息：当前可通过 App 内账号入口管理登录状态；未来会提供 account.useknowme.com
  • 邮件订阅记录：发邮件 contact@useknowme.com

5.2 数据删除权

  • 笔记内容：直接在你电脑上删除（我们没有副本）
  • 账号 / License 数据：发邮件 contact@useknowme.com，我们会在 30 天内处理
  • 邮件订阅：邮件底部有退订链接

5.3 数据导出权

  • 笔记内容：App 内 Settings → 导出 → 选 Markdown / JSON / SQLite 备份
  • 此功能永久免费，无论你是否订阅 Pro

6. 安全措施

数据类型 安全措施
你的笔记 ✅ 本地 SQLite + 你自己的磁盘加密（FileVault / BitLocker）
账号 / License 数据 ✅ Cloudflare D1 / KV（Cloudflare 平台级安全 + 最小化存储）
未来云同步（2027-07-07 后） ✅ 端到端加密（AES-256-GCM + 你的密码派生密钥）

7. Cookies

useknowme.com 网站仅使用必要 cookies：

  • Cloudflare 的 __cf_bm（DDoS 防护，30 分钟过期）
  • 暗黑模式偏好（localStorage，仅你浏览器内）

我们不使用： - ❌ Google Analytics 或同类追踪 - ❌ Facebook Pixel 等社交追踪 - ❌ 跨站追踪 cookie

8. 儿童隐私

KnowMe 不是为 13 岁以下儿童设计的。如发现 13 岁以下用户数据，我们会立即删除。

9. 变更通知

当此政策有重大变更时： - ✅ 提前 30 天在 useknowme.com 顶部 banner 公告 - ✅ 邮件通知所有注册邮箱（如你订阅过） - ✅ Git 提交记录公开，可追溯每次变更
