# 9. 快捷键速查

> macOS 用 `Cmd`，Windows / Linux 用 `Ctrl`。

## 9.1 标签页

| 快捷键 | 功能 |
|---|---|
| `Cmd+T` | 新建空白 tab |
| `Cmd+W` | 关闭当前 tab |
| `Cmd+Shift+T` | 重开最近关闭的 tab（最多 10 条历史）|
| `Cmd+1` ~ `Cmd+9` | 切换到第 N 个 tab |
| `Cmd+Tab` 内置 | 系统级（不归 KnowMe 管）|

## 9.2 编辑模式

| 快捷键 | 功能 |
|---|---|
| `Cmd+E` | Live ↔ Edit 切换 |
| `Cmd+Shift+E` | 切到 Reading |
| `Cmd+S` | 保存当前文件 |

## 9.3 列表 / 缩进

| 快捷键 | 功能 |
|---|---|
| `Tab` | 列表项缩进一级（同时重编号有序列表为 1.）|
| `Shift+Tab` | 列表项减少一级 |
| `Enter` | 列表内换行 + 自动续 marker（连续两次 Enter 退出列表）|

## 9.4 搜索 / 命令

| 快捷键 | 功能 |
|---|---|
| `Cmd+P` | 在 sidebar 搜索框聚焦 |
| `Cmd+Shift+F` | 全局笔记搜索 overlay |
| `Cmd+Shift+P` | 命令面板（计划中，v1.x）|

## 9.5 sidebar

| 快捷键 | 功能 |
|---|---|
| `Cmd+\` | 折叠 / 展开左 sidebar |
| `Cmd+Shift+\` | 折叠 / 展开右 STATUS 栏（计划中）|

## 9.6 Markdown 格式

| 快捷键 | 功能 |
|---|---|
| `Cmd+B` | 粗体 `**...**` |
| `Cmd+I` | 斜体 `*...*` |
| `Cmd+K` | 插入链接 `[text](url)` |

## 9.7 设置

| 快捷键 | 功能 |
|---|---|
| `Cmd+,` | 打开设置 overlay |
| `Esc` | 关闭设置 / 任何 overlay |

## 9.8 编辑器内查找替换

| 快捷键 | 功能 |
|---|---|
| `Cmd+F` | 打开查找面板（顶部）|
| `Cmd+Alt+F` | 打开查找替换面板 |
| `Cmd+G` / `Cmd+Shift+G` | 下一个 / 上一个匹配 |
| `Enter` / `Shift+Enter` | 下一个 / 上一个匹配（焦点在搜索框时）|
| `Esc` | 关闭面板 |

面板内提供选项：`match case`（区分大小写）、`regexp`（正则）、`by word`（全词匹配）。

## 9.9 编辑器其它默认快捷键

KnowMe 编辑器基于 CodeMirror 6，常用编辑快捷键：

| 快捷键 | 功能 |
|---|---|
| `Cmd+Z` / `Cmd+Shift+Z` | 撤销 / 重做 |
| `Cmd+A` | 全选 |
| `Cmd+D` | 选中下一个匹配 |
| `Cmd+L` | 选中整行 |
| `Cmd+/` | 注释当前行（不一定对 markdown 有意义）|
