# FAQ + 排错

## 常见问题

### 数据是不是都在本地？知我会上传我的笔记吗？

**不会。** 所有笔记内容存在你本地的 SQLite + 文件。编辑、编译、导出全本地。联网只在你显式触发相关能力时发生：调用你配置的 LLM、检查更新、下载更新、登录账号、进入未来云端能力等。调用 LLM 用的是你**自己配置的 API key**,直连你选的服务商,知我不中转、不留存。详见[配置 LLM · 隐私说明](/start/llm-setup)。

### 不登录可以用吗？

可以。本地编辑、搜索、Wiki、知识图谱、导出、BYOK AI 和本地 MCP 都不强制登录。账号用于进入 Agent Fleet（团队模式）以及未来云同步、分享、Pro / Team 权益等能力。详见[账号与登录](/start/account)。

### BYOK 是什么意思？

Bring Your Own Key——你自带 LLM API key。知我不内置付费模型额度,你用自己的 key(OpenAI / Anthropic / Google / 国产模型 / 本地模型),费用直接和服务商结算。

### 公测后哪些功能免费？

公测期(2026-06-06 → 2027-07-06)全功能免费。2027-07-07 Pro 上线后,**本地核心功能继续永久免费**。编辑器、Markdown/JSON 导出、本地知识库、BYOK、本地 LLM、基础图谱、基础 MCP 继续免费;Free 保留 6 种内置 Wiki Types 和已有类型数据的读取/导出;公测期开放的高级/预览能力可能进入 Pro。Pro 主要提供自定义 Wiki Types、云同步、托管 AI、云端索引、远程 API、后台 Agent Fleet、高级导出等能力。

### 早鸟 Pro 三年版是什么？

公测期内前 100 名,一次付费(价格待公布),享 **3 年本地 Pro 能力** + 这 3 年所有本地重大版本更新,早鸟价永远锁定。3 年后与普通用户一致(本地功能始终免费,Pro 新功能按需订阅)。

### 云同步包含在内吗？

**不包含。** 云同步、云端 AI、云端索引、远程 API 等托管服务有持续成本,对所有用户(含早鸟)都按订阅另付。早鸟三年版覆盖的是 Pro 的**本地能力**。

### 我能从 Obsidian 迁过来吗？

可以,**零迁移**。把 Obsidian 仓库**挂载**进知我即可,文件不挪、不污染 `.obsidian/`。详见[知识引擎 · 多挂载](/core/knowledge-engine)。

### 知我支持云同步吗？

当前 v0.5.x 不支持云同步。账号系统是未来 E2EE 云同步、分享和团队协作的身份地基,不代表你的笔记已经上传。备份 / 多设备方案:把 vault 放云盘(iCloud / Dropbox,注意多设备同时编辑会冲突),或用 git 手动版本管理。

### 怎么联系支持 / 报 bug？

`⌘,` → 关于 → 复制诊断信息,然后去 [GitHub Issues](https://github.com/Timeflys2018/KnowMe/issues) 提 issue(附复现步骤、期望 vs 实际、截图);或邮件 [contact@useknowme.com](mailto:contact@useknowme.com)。

## 故障排查

### macOS 提示"无法验证开发者"

Finder 找到 KnowMe.app → **右键 → 打开**(不要双击)→ 弹框点"打开"。之后双击即可正常启动。

### Windows / Linux 启动后空白

可能是 vault 路径含中文 / 空格 / Unicode。建议放纯英文路径,如 `C:\KnowMe-vault` 或 `~/knowme-vault`。

### KaTeX 公式没渲染

- 行内必须用 `$...$`(不能 `\(...\)`)。
- 块级必须独占段(前后空行),用 `$$...$$`。
- 语法错误时 KaTeX 显示红色 `[Math Error]`。

### 数据库 / 数据在哪？

见[配置中心 · 数据存储位置](/settings)。vault 完全可移植,整目录拷走即可。

### 自动保存 / 崩溃丢内容？

知我**默认自动保存**(输入停顿后写盘),也可随时 `⌘S`。仅最近未写盘的极短内容在崩溃时可能丢失,罕见。

### MCP server 连不上

- 确认启动命令路径正确(从设置复制最稳)。
- 确认 `--data-dir` 指向正确的数据目录。
- Claude Code 里用 `/mcp` 看连接状态;改了目录后重启 MCP 客户端。
- 详见 [MCP 接入](/integrate/mcp)。

### 注册 / 登录提示网络失败

如果你还在使用 v0.5.0,请升级到 v0.5.1 或更高版本。v0.5.1 已把账号服务切到稳定域名 `api.useknowme.com`。

### 收不到验证码

- 检查垃圾邮件和邮箱拼写。
- 等待重发冷却后再点重发。
- 短时间多次请求会触发限流,稍后再试即可。

### 为什么提示操作太频繁？

注册、登录、验证码、找回密码都有频率限制,用于保护邮箱发送额度和防暴力破解。等待几分钟后再试。

### 我误删了笔记

单文件删除进系统回收站可找回;若选了"永久删除"则暂无 in-app 撤销(vault 在 git 下可 `git checkout` 恢复)。
