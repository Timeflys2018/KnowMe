# 快捷键速查

> macOS 用 `⌘`,Windows / Linux 用 `Ctrl`。

## 导航 / 搜索

| 快捷键 | 功能 |
|---|---|
| `⌘K` | 全文搜索 |
| `⌘\` | 折叠 / 展开左 sidebar |
| `⌘,` | 打开设置 |
| `Esc` | 关闭设置 / 任何 overlay |

## 标签页

| 快捷键 | 功能 |
|---|---|
| `⌘T` | 新建 tab |
| `⌘W` | 关闭当前 tab |
| `⌘⇧T` | 重开最近关闭的 tab |
| `⌘1` ~ `⌘9` | 切换到第 N 个 tab |

## 编辑模式

| 快捷键 | 功能 |
|---|---|
| `⌘E` | Live ↔ Edit 切换 |
| `⌘⇧E` | 切到 Reading |
| `⌘S` | 立即保存(默认已自动保存) |

## 列表 / 缩进

| 快捷键 | 功能 |
|---|---|
| `Tab` / `Shift+Tab` | 列表项缩进 / 减一级 |
| `Enter` | 续 marker(连按两次退出列表) |

## Markdown 格式

| 快捷键 | 功能 |
|---|---|
| `⌘B` / `⌘I` | 粗体 / 斜体 |
| `⌘Z` / `⌘⇧Z` | 撤销 / 重做 |

## 查找替换

| 快捷键 | 功能 |
|---|---|
| `⌘F` | 查找 |
| `⌘⌥F` | 查找替换 |
| `⌘G` / `⌘⇧G` | 下一个 / 上一个匹配 |
