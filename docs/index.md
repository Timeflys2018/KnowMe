## 一句话定位

> KnowMe 是一个本地的、对标 Obsidian 的 markdown 笔记软件，多了：
>
> 1. **多通道导出**：微信公众号 / 小红书（计划中）等渠道一键发布
> 2. **AI 知识图谱**：自动从笔记里提取 entities、concepts、sources，建立 Wiki
> 3. **MCP 兼容**：Claude / Cursor / 任意 AI agent 都能直接读取你的笔记库

## 按场景找文档

::: tip 我从 Obsidian / 其他笔记迁过来
不用搬家。直接把现有仓库**挂载**进 KnowMe，文件不挪、零迁移。
→ [第 5 章 · 多标签 · 多挂载](/05-tabs-and-mounts)
:::

::: tip 我想让 Claude Code / Cursor 读我的笔记
配置 MCP，让任意 AI agent 直接查询你的本地知识库。
→ [第 7 章 · MCP · AI Agent](/07-mcp-and-agents)
:::

::: tip 我想把笔记发到微信公众号
一键样式化导出（含 KaTeX 公式、Mermaid 图），复制粘贴即用。
→ [第 6 章 · 微信公众号导出](/06-export-wechat)
