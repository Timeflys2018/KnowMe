# 小队（开发中）

::: warning 即将推出
小队 tab 当前在应用里**暂置灰下线（开发中）**，尚未对用户开放。下面描述的是即将推出的形态。以实际版本为准。
:::

小队是 agent 的**编组** —— 把几个 agent 组织成一队，并在消息 tab 用群聊式聚合视图看这一队的动向。

## 组织小队（规划）

- 建队 / 改名 / 删队。
- 加减成员（成员是通讯录里的 agent 模板引用）。
- 设队长（角色标记）。

## 群聊式聚合视图（规划）

选中一个小队 → 把该小队成员**各自独立会话**的消息，按时间归并成一条**只读时间线投影**，每条消息标注是哪个 agent 说的 —— 像看一个群的聊天记录。

::: tip 聚合视图 ≠ 聚合会话
小队是一个**观察视角**，不是一个共享会话。每个成员仍是独立的 1:1 会话；聚合视图只是把它们的消息按时间摆在一起看。会话永远不属于某个小队。
:::

## 边界（设计约束）

- 群聊聚合是**只读**的 —— 没有群输入框，不能在聚合视图里向全队发消息。
- **无 agent 间自动派发 / 自动 handoff / 队长自动编排** —— 队长是一个角色标记。
- 自主多 agent 编排属更远期能力，见 [Agent Fleet 总览 · 规划中](/integrate/agent-fleet#规划中-尚未-ship)。
